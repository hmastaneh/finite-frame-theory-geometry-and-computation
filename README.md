## Overview

This repository contains the lecture notes, computational laboratories, and supporting materials for the course

**Finite Frame Theory: Geometry and Computation**

developed for students at **Persian Gulf University**.

The project is designed as a modern introduction to finite frame theory, combining rigorous mathematics with geometric intuition and interactive computation. Throughout the course, students explore the theory of finite frames while learning how to investigate mathematical concepts using Python and Jupyter notebooks.

Unlike traditional lecture notes, this project integrates theoretical development with computational experiments, visualizations, and numerical algorithms.

---

## Features

- Comprehensive lecture notes
- Interactive Jupyter notebooks
- Computational laboratories for every chapter
- Numerical experiments using Python
- Geometric visualizations of frame constructions
- Exercises and programming assignments
- Open-source educational resource

## Software Requirements

The notebooks require

- Python 3.11+
- NumPy
- SciPy
- Matplotlib
- JupyterLab

Install all required packages with

```bash
pip install -r requirements.txt
```
